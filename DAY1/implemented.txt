2. Intent Detection (Rule + ML + LLM)

Functions:
detect_intent_rule_based()
detect_intent_ml()
detect_intent_llm()
detect_intent(mode="...")

Modes:
rule   → keyword rules (fast, deterministic)
ml     → Naive Bayes classifier (learned patterns)
llm    → Gemini-based classification
hybrid → rule -> ml -> llm fallback

Important update:
"joke" is removed from hardcoded rule checks.
Joke intent is now detected by ML or LLM.

Examples:
"add 5 and 10" + mode=rule   → math
"tell me a joke" + mode=rule → unknown
"tell me a joke" + mode=ml   → joke
"weather in tokyo" + mode=hybrid → weather

Runtime commands:
intent-mode: rule
intent-mode: ml
intent-mode: llm
intent-mode: hybrid
detect-intent: your text

Concept:
multiple intent strategies in one NLP system
